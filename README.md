# IJayaprakash.in

Personal website for [http://ijayaprakash.in/](http://ijayaprakash.in/) built with [Jekyll](http://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## Requirements

```sh
gem install github pages
```

## Usage

```sh
jekyll serve
```
